# unisol
---

## 🚀 프로젝트 개요

Canvas LMS는 웹 기반으로 제공되어 과제 확인 및 제출을 위해 브라우저를 별도로 사용해야 합니다.  
이 프로젝트는 이러한 불편함을 해결하기 위해, **VSCode 환경에서 과제 조회 및 제출을 직접 수행할 수 있도록 지원하는 Extension**입니다.
사용자는 IDE 내에서 수강 과목 조회, 과제 확인, 과제 제출까지 한 번에 처리할 수 있습니다.


## ✨ 주요 기능

### 📖 수강 과목 조회
- 현재 수강 중인 강의 목록 조회
- 강의 선택을 통한 과제 목록 확인

### 📝 과제 확인
- 과제 목록 및 상세 정보 제공
  - 과제 이름
  - 과제 설명
  - 마감일
  - 배점
  - 제출 방식

### 📤 과제 제출
- VSCode 내에서 과제 제출 가능
- 파일 업로드 기반 제출 지원
- UI를 통한 직관적인 제출 인터페이스 제공
